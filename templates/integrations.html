Configurazione necessaria

Per collegare Google Fit devi creare un progetto su Google Cloud Console:

  1. Vai su console.cloud.google.com
  2. Crea un nuovo progetto o selezionane uno esistente
  3. Vai a "API e Servizi" → "Libreria" e abilita Fitness API
  4. Vai a "API e Servizi" → "Credenziali" → "Crea credenziali" → "ID client OAuth"
  5. Tipo applicazione: Applicazione Web
  6. URI di reindirizzamento autorizzato: {{ config.GOOGLE_REDIRECT_URI }}
  7. Copia il Client ID e Client Secret nel file .env

Dopo aver configurato il .env, ricarica questa pagina.

Come funziona

  1. Collega il tuo orologio o dispositivo all'app Health Connect sul telefono (preinstallata su Android 14+).
  2. In Health Connect, concedi a Google Fit di leggere i dati dal dispositivo.
  3. Clicca "Collega con Google" sopra per autorizzare Ajò a leggere i dati.
  4. Clicca "Sincronizza ora" per importare allenamenti e peso degli ultimi 7 giorni.
